== Description ==

 Plugin allow smoothly scroll your #anchor links.

== Installation ==

1. Upload `/wp-link-scroller` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin through the 'Settings - WP Link Scroller' menu in WordPress
